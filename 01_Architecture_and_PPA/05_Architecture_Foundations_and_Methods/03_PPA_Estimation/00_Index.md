# Architecture Foundations and Methods › Power, Performance, and Area (PPA) Estimation

> **Abbreviation key — skim now and return as needed:** register-transfer level (RTL).

**Plain-language purpose:** Produce useful early power, performance, and area (PPA) ranges before detailed implementation exists, while making uncertainty visible.

## Terms introduced here

| Term | Meaning |
|---|---|
| proxy | measurable structure used to estimate something not yet implemented |
| calibration | corrects a model using trusted reference observations |
| sensitivity | change in output caused by a change in one input |
| uncertainty interval | credible range rather than false point precision |
| Pareto frontier | choices not dominated in every objective |

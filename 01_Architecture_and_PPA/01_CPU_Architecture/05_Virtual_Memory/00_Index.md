# Central Processing Unit (CPU) Architecture › Virtual Memory

> **Abbreviation key — skim now and return as needed:** translation lookaside buffer (TLB); input-output memory management unit (IOMMU); virtual address (VA); physical address (PA).

**Plain-language purpose:** Explain how software addresses become protected physical memory accesses for CPU cores and system devices.

## Terms introduced here

| Term | Meaning |
|---|---|
| virtual address (VA) | address generated by software |
| physical address (PA) | address used by caches, memory, or devices after translation |
| page | fixed-size unit of address mapping and protection |
| translation lookaside buffer (TLB) | cache of recent virtual-to-physical mappings |
| page-table walker | hardware that reads page tables after a TLB miss |
| input-output memory management unit (IOMMU) | translation and protection for device requests |

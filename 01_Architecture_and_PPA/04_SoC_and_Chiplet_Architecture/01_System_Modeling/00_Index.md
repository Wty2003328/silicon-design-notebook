# System-on-Chip (SoC) and Chiplet Architecture › System Modeling

> **Abbreviation key — skim now and return as needed:** central processing unit (CPU); graphics processing unit (GPU); neural processing unit (NPU); power, performance, and area (PPA); dynamic voltage and frequency scaling (DVFS).

**Plain-language purpose:** Combine architecture-owned blocks into one product model with shared bottlenecks, power limits, and thermal feedback.

## Terms introduced here

| Term | Meaning |
|---|---|
| system on chip (SoC) | processors, memory, fabrics, and devices integrated as one chip |
| composition | builds a system estimate from calibrated block models |
| contention | requesters delay one another at a shared resource |
| dynamic voltage and frequency scaling (DVFS) | changes voltage/frequency while running |
| thermal feedback | temperature changes leakage and allowed operating point |
